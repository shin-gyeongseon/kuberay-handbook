    5.5 Ray AutoscalerмҷҖ Kubernetes HPA м°Ёмқҙ л°Ҹ м—°лҸҷ¶

    RayлҠ” мһҗмІҙм ҒмңјлЎң нҒҙлҹ¬мҠӨн„° мҳӨнҶ мҠӨмјҖмқјл§Ғ кё°лҠҘмқ„ м ңкіөн•ҳм§Җл§Ң, мҝ лІ„л„ӨнӢ°мҠӨ нҷҳкІҪм—җм„ңлҠ” мў…мў… Kubernetes HPA(Horizontal Pod Autoscaler)лӮҳ KarpenterмҷҖ н•Ёк»ҳ мӮ¬мҡ©лҗҳкё°лҸ„ н•©лӢҲлӢӨ. мқҙ л¬ём„ңм—җм„ңлҠ” Ray AutoscalerмҷҖ Kubernetes HPAмқҳ м°ЁмқҙлҘј лӘ…нҷ•н•ҳкІҢ кө¬л¶„н•ҳкі , л‘җ л°©мӢқмқ„ м–ҙл–»кІҢ н•Ёк»ҳ нҷңмҡ©н•  мҲҳ мһҲлҠ”м§Җ м •лҰ¬н•©лӢҲлӢӨ.

    вң… Ray AutoscalerлһҖ?¶

    Ray AutoscalerлҠ” Ray Head л…ёл“ңм—җ лӮҙмһҘлҗң нҒҙлҹ¬мҠӨн„° л ҲлІЁ мҠӨмјҖмқјл§Ғ м»ҙнҸ¬л„ҢнҠёмһ…лӢҲлӢӨ. мӮ¬мҡ©мһҗмқҳ мӣҢнҒ¬лЎңл“ң(task, actor л“ұ)к°Җ лҠҳм–ҙлӮҳл©ҙ worker podлҘј мһҗлҸҷмңјлЎң м¶”к°Җн•ҳкұ°лӮҳ к°җмҶҢмӢңнӮӨлҠ” м—ӯн• мқ„ мҲҳн–үн•©лӢҲлӢӨ.

    • лҸҷмһ‘ кё°мӨҖ: Ray лӮҙл¶Җмқҳ task нҒҗ / лҰ¬мҶҢмҠӨ лҢҖкё°м—ҙ
    • мҠӨмјҖмқј лҢҖмғҒ: Ray worker л…ёл“ң
    • мӢӨн–ү мң„м№ҳ: Ray head pod лӮҙл¶Җ (лі„лҸ„ м»ЁнҠёлЎӨлҹ¬ м•„лӢҳ)
    • мһ‘лҸҷ л°©мӢқ: RayCluster.spec.enableInTreeAutoscaling = true м„Өм • мӢң мһҗлҸҷ нҷңм„ұнҷ”лҗЁ

    м°ёкі : Rayмқҳ AutoscalerлҠ” лӮҙл¶Җм ҒмңјлЎң resource demandлҘј кё°л°ҳмңјлЎң лҸҷмһ‘н•ҳлҜҖлЎң, мӮ¬мҡ©мһҗк°Җ м§Ғм ‘ лҰ¬мҶҢмҠӨ мҡ”мІӯ/н• лӢ№мқ„ м ңм–ҙн•ҳм§Җ м•Ҡм•„лҸ„ лҗ©лӢҲлӢӨ.

    вң… Kubernetes HPAлһҖ?¶

    Kubernetes HPAлҠ” нҠ№м • DeploymentлӮҳ Podмқҳ CPU/л©”лӘЁлҰ¬ мӮ¬мҡ©лҘ  лҳҗлҠ” мӮ¬мҡ©мһҗ м •мқҳ metricмқ„ кё°мӨҖмңјлЎң replica мҲҳлҘј мһҗлҸҷ мЎ°м Ҳн•ҳлҠ” Kubernetesмқҳ н‘ңмӨҖ мҳӨнҶ мҠӨмјҖмқјлҹ¬мһ…лӢҲлӢӨ.

    • лҸҷмһ‘ кё°мӨҖ: CPU/л©”лӘЁлҰ¬ мӮ¬мҡ©лҹү лҳҗлҠ” Custom Metric
    • мҠӨмјҖмқј лҢҖмғҒ: Replica к°ңмҲҳ (ex. Web App, API м„ңлІ„ л“ұ)
    • мӢӨн–ү мң„м№ҳ: Kubernetes Control Plane
    • кё°л°ҳ лҰ¬мҶҢмҠӨ: metrics-server, Prometheus Adapter л“ұ н•„мҡ”

    рҹ”Қ Ray Autoscaler vs HPA л№„көҗ¶

    н•ӯлӘ© Ray Autoscaler Kubernetes HPA
    м Ғмҡ© лҢҖмғҒ Worker Pod мқјл°ҳ Deployment
    лҸҷмһ‘ кё°мӨҖ лҰ¬мҶҢмҠӨ лҢҖкё°м—ҙ (Task/Actor) лҰ¬мҶҢмҠӨ мӮ¬мҡ©лҘ  (CPU/Memory)
    мһ‘лҸҷ мң„м№ҳ Ray Head лӮҙл¶Җ Kubernetes Master
    м—°лҸҷ л°©мӢқ RayCluster мҠӨнҺҷ кё°л°ҳ лі„лҸ„ Metric + HPA лҰ¬мҶҢмҠӨ мғқм„ұ н•„мҡ”
    лӘ©м Ғ нҒҙлҹ¬мҠӨн„° мҠӨмјҖмқјл§Ғ м• н”ҢлҰ¬мјҖмқҙм…ҳ л ҲлІЁ мҠӨмјҖмқјл§Ғ

    рҹ”§ н•Ёк»ҳ мӮ¬мҡ©н•ҳлҠ” кІҪмҡ°¶

    • Ray AutoscalerлҠ” нҒҙлҹ¬мҠӨн„° нҷ•мһҘ/м¶•мҶҢм—җ м§‘мӨ‘
    • Kubernetes HPAлҠ” Serve ReplicaлӮҳ RayJobмқҳ Pod мҲҳ мЎ°м Ҳм—җ м Ғн•©

    мҳҲмӢң: Ray ServeмҷҖ HPAлҘј н•Ёк»ҳ мӮ¬мҡ©н•ҳлҠ” мӢңлӮҳлҰ¬мҳӨ¶

    apiVersion: autoscaling/v2
kind: HorizontalPodAutoscaler
metadata:
  name: ray-serve-hpa
spec:
  scaleTargetRef:
    apiVersion: apps/v1
    kind: Deployment
    name: ray-serve-replica
  minReplicas: 1
  maxReplicas: 5
  metrics:
    - type: Resource
      resource:
        name: cpu
        target:
          type: Utilization
          averageUtilization: 60

    рҹ’Ў м •лҰ¬¶

    • Ray AutoscalerлҠ” нҒҙлҹ¬мҠӨн„° лӢЁмң„мқҳ мҠӨмјҖмқјл§Ғмқ„ мһҗлҸҷнҷ”н•©лӢҲлӢӨ.
    • Kubernetes HPAлҠ” м„ңл№„мҠӨ нҳ№мқҖ л Ҳн”ҢлҰ¬м№ҙ лӢЁмң„мқҳ мҠӨмјҖмқјл§Ғмқ„ мІҳлҰ¬н•©лӢҲлӢӨ.
    • л‘ҳмқҖ мӨ‘ліөлҗҳлҠ” к°ңл…җмқҙ м•„лӢҲлқј м„ңлЎң ліҙмҷ„м Ғмһ…лӢҲлӢӨ.
    • мӮ¬мҡ©мһҗмқҳ м„ңл№„мҠӨ кө¬мЎ°м—җ л”°лқј м Ғм ҲнһҲ л¶„лҰ¬ мӮ¬мҡ©н•ҳкұ°лӮҳ лі‘н–ү м Ғмҡ©н•  мҲҳ мһҲмҠөлӢҲлӢӨ.
